## ✦ About

**Glass** is a premium glassmorphism HTML template for modern brands, startups, and digital products. Features frosted glass cards, animated fluid gradients, and built-in dark mode.

---

## ✦ Features

| Feature | Description |
|---------|-------------|
| 🪟 **Frosted Glass** | CSS backdrop-filter cards with blur and translucency |
| 🌈 **Fluid Gradients** | Animated gradient backgrounds with drifting orbs |
| 🌙 **Dark Mode** | Built-in toggle with localStorage persistence |
| 🖱️ **Orb Parallax** | Floating glass orbs respond to mouse movement |
| 🔍 **Scroll Reveals** | Cards and sections fade up on scroll |
| 📱 **Responsive** | Mobile, tablet, desktop |
| 🎨 **CSS Variables** | One-file color theming |
| 🚀 **Zero Deps** | No frameworks, no build tools |
