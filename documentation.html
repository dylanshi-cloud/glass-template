Glass Template

Premium Glassmorphism Template — Documentation

📦 What's Included

  • index.html — Full template (Hero, Showcase, Features, Pricing, Contact, Footer)
  • css/style.css — Complete stylesheet with CSS variables + dark mode
  • js/main.js — All interactions (dark mode toggle, scroll reveals, mobile nav)

🎨 How to Customize

Step 1: Brand Colors

Open css/style.css and edit the :root variables:

  --accent: #6366F1;        /* Main brand color (purple) */
  --accent-light: #A5B4FC;  /* Light variant */
  --accent-dark: #4F46E5;   /* Dark variant */

Dark mode colors are in [data-theme="dark"].

Step 2: Brand Text

Open index.html and search for CUSTOMIZE. Each section has a comment marking what to edit.

Step 3: Dark Mode

Dark mode is built-in. The theme toggle button saves user preference to localStorage. To remove it, delete the .theme-toggle button from the HTML and the dark mode JS from main.js.

Step 4: Set Up Order Email

Open js/main.js and find the contact form — update the email address to yours.
